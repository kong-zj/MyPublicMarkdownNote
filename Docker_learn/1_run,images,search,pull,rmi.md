## search

在远程仓库Docker Hub上搜索镜像
![](2022-12-15-13-34-15.png)

### docker search -f stars=30 tomcat

过滤镜像，只要stars超过30的
![](2022-12-15-13-37-20.png)

### docker search -f stars=30 --no-trunc tomcat

参数```--no-trunc```表示不要省略说明
![](2022-12-15-13-39-10.png)
