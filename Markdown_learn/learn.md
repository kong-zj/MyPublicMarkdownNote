### 标题

### 段落

one line

### 代码块

三个`（<kbd>Tab</kbd>上方的按键）围住，可指定语言

```python
print('1')
```

```javascript
var s = "dssfnljs";
alert(s);
```

### 转义字符

要显示原本用于格式化 Markdown 文档的字符，请在字符前面添加反斜杠字符 \

\*\#

### 数学公式

<kbd>Ctrl</kbd> + <kbd>M</kbd>，按两次

$$
\lim_{x \to \infin}\frac{sin(t)}{x}=1
$$

### 表格

| 孔  | 子  | 健  |
| --- | --- | --- |
| 1.3 | 2.4 | 4.1 |

<kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>F</kbd>，表格格式化（也是代码格式化的快捷键）

 ### 专注模式
